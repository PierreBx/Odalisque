# flutter_grist_widgets

A Flutter library for visualizing and editing data tables stored in Grist.

## Overview

`flutter_grist_widgets` makes it incredibly easy to display and edit data from Grist tables in your Flutter applications. With minimal configuration, you can render Grist data in various formats:

- **Tabular views** - Display data in traditional table format
- **Form views** - Present data as editable forms
